# Graphing-tool
A tool for plotting graphs of multivariate functions. Made using c++ / OpenGL by github.com/Milan7843.

# Controls
 - W to move forward
 - S to move backward
 - A to move left
 - D to move right
 - E to move up
 - Q to move down
 - F to enable/disable settings overlay
 - Mouse to look around
Settings are inside the src/main.cpp file.

# To run
Simply run the Graphing-tool.exe inside of the builds folder.
